About Pokémon App

Welcome! In this Web Application, which it was built with HTML, CSS & JavaScript you can load Pokémon data from an external API, which it enables the viewing of data points in detail.

Click on a Pokémon's name to see more details on that selected Pokémon. Use the search bar to find a specific Pokémon by name. Happy Searching!
